# 板块实时热力图 MVP 设计说明

## 目标

板块实时热力图不是简单的红绿涨跌图，而是用于快速判断：

1. 今天市场主线是什么。
2. 主力资金真正流向哪里。
3. 哪些板块只是龙头独涨，哪些板块已经形成扩散。
4. 哪些板块可以进一步映射到 ETF / 指数基金观察池。

## 当前实现

当前版本是纯静态前端 Demo，入口为：

```text
index.html
```

核心文件：

```text
src/main.js                  # 页面交互、指标计算、热力图渲染
src/styles.css               # 页面视觉样式
src/data/mock-sectors.js     # 行业 / 概念 Mock 数据
src/services/sector-api.js   # 数据适配层，后续替换为真实 API
```

## 展示方式

### 主视图

采用「卡片矩阵热力图」而不是传统表格：

- 面积表示关注度或规模。
- 颜色表示涨跌、资金或综合热度。
- 边框表示主力资金方向。
- 卡片内展示板块名、涨跌 / 热度、主力净流入、上涨下跌家数、领涨股。

### 三种模式

| 模式 | 面积 | 颜色 | 价值 |
| --- | --- | --- | --- |
| 涨跌热力 | 可切换，默认成交额 | 涨跌幅 | 快速看市场强弱 |
| 资金热力 | 可切换，默认成交额 | 主力净流入占比 | 看资金真实流向 |
| 综合热度 | 可切换，默认成交额 | hotScore | 识别今日主线 |

### 三种面积口径

| 口径 | 适用场景 |
| --- | --- |
| 成交额 | 默认推荐，看实时市场关注度 |
| 资金规模 | 看资金集中攻击方向 |
| 总市值 | 看市场结构，但不适合默认 |

## 指标计算

### 内部扩散度

```text
riseRatio = upCount / (upCount + downCount) * 100
```

用于判断板块是全面扩散，还是少数龙头独涨。

### 综合热度分

当前前端临时计算：

```text
hotScore = 50
  + changePct * 6
  + mainNetInRatio * 4
  + riseRatio * 0.18
  + turnoverRate * 1.6
  + leadingStockChangePct * 0.5
```

后续建议迁移到后端，统一规则并支持回测。

## 异动标签

| 标签 | 当前规则 |
| --- | --- |
| 强势共振 | 涨跌幅 > 2%，主力净流入 > 0，扩散度 > 70% |
| 资金抢筹 | 涨幅不高，但主力净占比 > 2% |
| 高位分歧 | 涨幅 > 1%，但主力净流出 |
| 板块退潮 | 跌幅 < -1%，主力净流出，扩散度 < 45% |
| 龙头独涨 | 领涨股涨幅 > 7%，扩散度 < 55% |
| 全面扩散 | 扩散度 > 80%，成交额 > 350 亿 |
| 放量 | 换手率 > 4.5% |

## 后续真实 API 接入

推荐接口契约：

```text
GET /api/sector/heatmap?type=industry&period=today&metric=change
GET /api/sector/heatmap?type=concept&period=today&metric=change

GET /api/sector/:code/stocks
```

前端已经预留 `src/services/sector-api.js`，后续将 Mock 替换为真实请求即可。
